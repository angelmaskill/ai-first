# Legacy Fullstack Example

This example represents a brownfield full-stack repository layout that can be adopted by the AI-first scaffold.

Suggested flow:

```bash
node dist/index.js adopt ./examples/legacy-fullstack
node dist/index.js guide ./examples/legacy-fullstack
node dist/index.js task ./examples/legacy-fullstack "inventory-sync" "Coordinate frontend and backend inventory updates"
node dist/index.js review ./examples/legacy-fullstack
node dist/index.js sync ./examples/legacy-fullstack
```
